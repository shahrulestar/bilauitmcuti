# Bila UiTM Cuti? - Academic Calendar

> Last updated: rolling updates

## Overview
Bila UiTM Cuti? is an interactive UiTM calendar web app for academic calendar schedules. It is not an official UiTM product; it is built by Shahrul Estar for students and staff who need quick access to registration dates, lecture schedules, examination periods, and breaks.

Base URL: https://bilauitmcuti.com

## Key Features
- **View Modes**: List view and Grid view for browsing the calendar
- **Program Selection**: All programs, or by program: Foundation/Professional, Pre-Diploma, Diploma, Diploma (Part-Time), Bachelor, Bachelor (Part-Time), Master, PhD
- **Group-based Schedules**: Group A (Foundation/Professional) and Group B (Pre-Diploma onwards) with different schedules
- **Activity Filters**: Toggle visibility of Registration, Lecture, Short Semester, Intersession Classes, Examination, Others Exams, and Break
- **KKT State Support**: Special dates for Kedah, Kelantan, and Terengganu
- **Theme**: Light and Dark mode
- **Responsive Design**: Works on desktop and mobile
- **PWA**: Installable app with offline access via service worker
- **Persistent Preferences**: User selections (program, view, filters, theme) saved via localStorage and cookie
- **AI Chat**: Ask questions about UiTM academic calendar via AI assistant at /chat

## FAQ

### Bila UiTM buka semester?
Tarikh pembukaan semester UiTM berbeza mengikut program dan sesi. Layari bilauitmcuti.com untuk melihat jadual lengkap pendaftaran dan kuliah bagi Foundation, Pre-Diploma, Diploma, Bachelor, Master dan PhD.

### Bila peperiksaan UiTM?
Tarikh peperiksaan UiTM bergantung kepada program dan sesi. Lihat kalendar akademik interaktif di bilauitmcuti.com untuk tarikh peperiksaan tepat termasuk peperiksaan khas dan EET.

### Apa itu Semester Pendek UiTM?
Semester Pendek adalah sesi pengajian singkat yang diadakan antara semester biasa. Ia membolehkan pelajar mengambil kursus tambahan. Jadual Semester Pendek boleh dilihat di bilauitmcuti.com.

### Macam mana nak guna Bila UiTM Cuti?
Layari bilauitmcuti.com, pilih program anda (Foundation, Diploma, Bachelor, dll), dan lihat kalendar dalam paparan grid atau senarai. Anda juga boleh install sebagai PWA untuk akses lebih pantas.

### Bila cuti semester UiTM?
Cuti semester UiTM berbeza mengikut kumpulan program dan sesi. Semak tarikh cuti semester lengkap di bilauitmcuti.com dengan memilih program anda.

## Data Structure
Activities are defined by: month/year, program group (A or B), activity type (registration, lecture, examination, break, other), start and end dates, optional KKT state variants. Types include Short Semester and Intersession Classes where applicable. Each activity may have a programType (PreDiploma, Diploma, DiplomaPartTime, Bachelor, BachelorPartTime, Master, PhD) or allStudents flag for all students.
