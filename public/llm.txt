# Bila UiTM Cuti? - Academic Calendar 2026

## Overview
Bila UiTM Cuti? is an interactive academic calendar web app for UiTM (Universiti Teknologi MARA) showing scheduled activities for 2026. It is not an official UiTM product; it is for students and staff who need quick access to registration dates, lecture schedules, examination periods, and breaks.

Base URL: https://cutiuitm.xyz

## Key Features
- **View Modes**: List view and Grid view for browsing the calendar
- **Program Selection**: All programs, or by program: Foundation/Professional, Pre-Diploma, Diploma, Diploma (Part-Time), Bachelor, Bachelor (Part-Time), Master, PhD
- **Group-based Schedules**: Group A (Foundation/Professional) and Group B (Pre-Diploma onwards) with different schedules
- **Activity Filters**: Toggle visibility of Registration, Lecture, Semester Pendek, Kuliah Intersesi, Examination, Others Exams, and Break
- **KKT State Support**: Special dates for Kedah, Kelantan, and Terengganu
- **Theme**: Light and Dark mode with smooth transitions
- **Responsive Design**: Works on desktop and mobile
- **PWA**: Installable app with offline access
- **Persistent Preferences**: User selections (program, view, filters, theme) saved via localStorage and cookie

## Pages and Content
- **/** — Home. All programs, grid view. Main entry point.
- **/list** — All programs, list view. Chronological list of activities.
- **/[program]** — Calendar grid for one program. Valid slugs: foundation-professional, pre-diploma, diploma, diploma-part-time, bachelor, bachelor-part-time, master, phd.
- **/[program]/list** — Calendar list view for one program. Same slugs as above.
- **/changelog** — Changelog: list of updates and improvements to the app, grouped by month.
- **/pwa** — How to install the app as PWA (Desktop, iOS, Android), offline features, and disclaimer that the app is not affiliated with UiTM.

## Data Structure
Activities are defined by: month/year, program group (A or B), activity type (registration, lecture, examination, break, other), start and end dates, optional KKT state variants. Types include Semester Pendek and Kuliah Intersesi where applicable.

## Technology
- Next.js (App Router), shadcn/ui, Tailwind CSS v4, React hooks, Geist font. Service worker for PWA.
