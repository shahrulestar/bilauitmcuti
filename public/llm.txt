# Bila UiTM Cuti - Academic Calenar

This file is intended for LLM indexing and summarization.
Primary site: https://bilauitmcuti.com
Project type: UiTM academic calendar web app for students.

## One-line summary
Bila UiTM Cuti helps UiTM students check academic dates (registration, lectures, exams, and breaks) by program and session without relying on multiple PDFs.

## What the app does
- Shows UiTM academic calendar data in grid and list views.
- Supports program-specific schedules from Foundation/Professional to PhD.
- Includes Group A and Group B schedule differences.
- Handles regional date variations for Kedah, Kelantan, and Terengganu.
- Provides an AI chat assistant for calendar and UiTM general questions.
- Links to Find My Internship for discovering internship opportunities from multiple sources across Malaysia.

## Intended users
- UiTM students who need fast semester planning.
- UiTM community members looking for a simpler calendar reference.

## Stack
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS + shadcn/ui + Radix UI
- Cloudflare Pages (deployment)
- Cloudflare Workers AI (chat)
- Service worker (PWA support)

## Data source
Calendar data comes from an upstream API origin configured by server environment (`CALENDAR_API_BASE`, default `https://api.bilauitmcuti.com`).
Client requests use same-origin API routes; upstream details stay server-side.

## Privacy and security notes
- No user accounts.
- No personal profile storage.
- Uses Cloudflare Analytics and Google Analytics for traffic/usage analytics.
- AI queries are stateless and not tied to user profiles.
- Security reporting: https://bilauitmcuti.com/feedback and hello@bilauitmcuti.com

## Main pages
- / : main calendar
- /list : list calendar view
- /chat : AI assistant
- /feedback : feedback and report form
- /sponsor : redirects to https://shahrulestar.com/sponsor
- /about : project details
- /internship : Find My Internship — discover internship opportunities from multiple sources across Malaysia

## Important context for LLMs
- This is not an official UiTM product.
- Use it as a convenience layer for calendar lookup, not as legal/official policy text.
- If date conflicts happen, official UiTM announcements should be treated as final source of truth.
